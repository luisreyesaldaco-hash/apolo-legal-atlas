# apolo-legal-atlas
Agente legal computable para el módulo de arrendamiento.

## Descripción
Apolo Legal Atlas es un prototipo que demuestra cómo estructurar y consultar información jurídica de forma computable. Este repositorio contiene la interfaz básica y los archivos necesarios para desplegar el MVP.

## Contenido
- Interfaz del agente
- Archivos base del proyecto
- Configuración lista para desplegar en Vercel

## Objetivo
Explorar un modelo inicial de asesoría legal computable.
